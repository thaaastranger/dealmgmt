Usage Examples

JavaScript API

Create buttons programmatically using the Button class:

// Create a button with JavaScript const button = new Button({ label: 'Click me', variant: 'fill', size: 'large', leadingIcon: '<svg>...</svg>', onClick: (event) => { console.log('Button clicked!'); } }); // Append to DOM document.getElementById('container').appendChild(button.render()); // Update button button.update({ label: 'Updated', disabled: true });

HTML Structure

Use semantic HTML with CSS classes:

<!-- Fill button --> <button class="btn btn-fill btn-large"> <span class="btn-icon"> <svg>...</svg> </span> <span class="btn-label">Label</span> </button> <!-- Outline button --> <button class="btn btn-outline btn-medium"> <span class="btn-label">Label</span> </button> <!-- Text-only button --> <button class="btn btn-text-only btn-small" disabled> <span class="btn-label">Label</span> </button>

Design Token System

2-Level Token Architecture

Raw tokens define base colors, semantic tokens map to component-specific usage.

Raw Tokens (Level 1)
--color-grey-black: #000000 --color-grey-white: #FFFFFF --color-grey-50: #BABABA --color-grey-20: #F7F7F7
Semantic Tokens (Level 2)
--button-fill-background --button-fill-content --button-outline-border --button-text-content
